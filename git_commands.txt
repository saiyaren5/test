远程仓库
创建SSH Key:
1,ssh-keygen -t rsa -C "你的邮箱地址"
2,记录.ssh目录位值
ssh:id_rsa(私钥) id_rsa.pub(公钥)

ssh -T git@github.com

创建远程仓库

关联远程仓库
git remote add origin 远程仓库地址

删除关联
git remote rm origin


推送本地内容至远程仓库
git push origin master

拉取程仓库至本地仓库
git pull origin master --allow-unrelated-histories

创建文件：
touch .gitignore

从零开发：
先有远程库，再有本地库
克隆：git clone 远程库地址

git add .
git commit * -m "xxx"

删除ssh key
rm -rf .ssh

git checkout -b 分支名

查看当前分支：
git branch

切换到master分支
git checkout master
把dev分支的成果合并到master
git merge dev

创建远程分支
git push --set-upstream origin xxxx
